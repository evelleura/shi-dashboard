# References

## Kajian Hasil Penelitian (Tabel 2.1 - 7 penelitian terdahulu)

### azkia
`[[ref:azkia2024]]` Azkia, Yekti, Caesaron (2024) - "Perancangan Dashboard Monitoring dan Controlling Kinerja Proyek pada PT. XYZ Menggunakan Metode Agile and Lean Development." Action Research Literate 8(9). EVM + Looker Studio + Google Sheets, 7 fitur (Project Overview, Financial, Timeline, Milestone, Resource, Performance, Risk). Black Box OK, UAT 86%.

### ernawan
`[[ref:ernawan2024]]` Ernawan (2024) - "Pemanfaatan Management Dashboard dalam Pengambilan Keputusan Strategis pada Perusahaan Bisnis Konstruksi (Studi Kasus PT. XYZ)." Jurnal Mirai Management 9(2). Dashboard manajemen -> KPI real-time, akurasi data naik, identifikasi masalah cepat. **Dikutip di Latar Belakang** sebagai bukti efektivitas -> `[[thesis]]` inspired_by.

### gledson
`[[ref:gledson2024]]` Gledson, Rogage, Thompson, Ponton (2024) - "Reporting on the Development of a Web-Based Prototype Dashboard for Construction Design Managers" (DSRM). Buildings 14(2). Digitalisasi koordinasi desain, analisis TQ + RFI.

### hakim
`[[ref:hakim2025]]` Hakim & Pradibta (2025) - "Pengembangan Fitur Dashboard dan Report pada Aplikasi Project Management PT Intelix." Jurnal Kendali Teknik dan Sains 3(4). Agile Kanban, 3 tabel dashboard + sub-menu report, evaluasi SUS "sangat sesuai".

### auliansyah
`[[ref:auliansyah2023]]` Auliansyah, Irawan, Ariwibisono (2023) - "Rancang Bangun Sistem Monitoring Manajemen Proyek Konstruksi Menggunakan Kurva-S." JATI 6(2). RAB + schedule + laporan mingguan -> Gantt + Kurva-S, hitung deviasi (-0.17%, -1.19%), multi-user 4 hak akses.

### iqbal
`[[ref:iqbal2024]]` Iqbal dkk. (2024) - "Sistem Manajemen Proyek pada Startup Jasa Pembuatan Aplikasi." Klik 5(2). Waterfall, web, manajemen tugas + Gantt + dashboard ringkasan. Black Box stabil.

### luthan
`[[ref:luthan2023]]` Luthan, Sitanggang, Syafriandi (2023) - "Reinventing Formulas for Construction Project Delay Index Due to Management and Production." HighTech and Innovation Journal 4(4). Rumuskan **MDI (Management Delay Index)** + **PDI (Production Delay Index)** -> dashboard Ms. Project, skor TELOS 8.26/10. Konsep delay-index manajemen-vs-produksi menjadi inspirasi `[[concept:escalation]]`.
